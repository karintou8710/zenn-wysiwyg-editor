# zenn-wysiwyg-editor

Zenn の記事を WYSIWYG で編集可能なエディタです。（非公式）
編集後はコピーボタンで対応するマークダウンのエクスポートが可能となります。

## 機能

ノード

- [x] 見出し
- [x] リスト
- [x] テキストリンク
- [ ] 画像
- [ ] テーブル
- [ ] コードブロック
- [x] 引用
- [ ] 脚注
- [x] 区切り線

マーク

- [ ] イタリック
- [ ] 太字
- [ ] 打ち消し線
- [ ] インラインコード

Zenn 独自ノード

- [ ] メッセージ
- [ ] アコーディオン
- [ ] リンクカード
- [ ] X（Twitter）のポスト
- [ ] YouTube
- [ ] GitHub
- [ ] GitHub Gist
- [ ] CodePen
- [ ] SlideShare
- [ ] SpeakerDeck
- [ ] Docswell
- [ ] JSFiddle
- [ ] CodeSandbox
- [ ] StackBlitz
- [ ] Figma
